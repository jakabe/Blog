So Many To Dos

So I've got a lot of things on my to do list. For about a month my top priority was building a better to do list. The outcome was my Task Board which can be found here. Below You will see a snippet of my current Task Board as I see it. Should anyone else choose to use this tool, it's worth noting that any lists and items you add will be saved to a JSON file locally. If you delete your cookies that list will be lost. I know, I need to add an export button to prevent this exact thing.
